# Macro Dashboard Backend

## Project Structure

- **src/**: Contains application source code.
  - **controllers/**: Handle business logic and incoming requests.
  - **models/**: Define data models.
  - **routes/**: Define API routes.
  - **services/**: Contains reusable business logic.
- **config/**: Configuration files, such as database connection.
- **.env**: Environment variables.
- **package.json**: Metadata and dependencies.

## Instructions
1. Install dependencies: `npm install`
2. Run the app: `node src/index.js`
